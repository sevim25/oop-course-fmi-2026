# 🏟 Футболно първенство 🏆

_Да се реализира система за управление на футболно първенство (лига)._

В лигата участват различни **отбори**.

Всеки отбор се характеризира с:
- име
- _може да добавите и други член-данни, в случая няма да задълбаваме_

Всеки отбор може да взима участие в няколко първенства.

Всеки участник в първенство се характеризира с:
- отбор
- статистика за участието му в конкретно първенство
- списък от мачове

Статистиката включва:
- изиграни мачове
- победи
- равенства
- загуби
- отбелязани голове
- допуснати голове
- точки

Важно:
- Един и същи отбор може да участва в повече от една лига.
- Статистиката е валидна само в рамките на конкретната лига.
- Всеки участник трябва да има достъп до мачовете, в които е участвал.


В лигата се играят множество **мачове**.

Всеки мач се характеризира с:
- уникален идентификатор
- домакин
- гост
- голове на домакина
- голове на госта
- информация дали мачът е изигран

Важно:
- Всеки мач е между точно два различни отбора.
- Един отбор може да участва в много мачове.
- Един мач може да бъде създаден:
  - без резултат
  - или директно с подаден резултат

Всяка **лига** се характеризира с:
- име
- списък от участници
- списък от мачове

### Функционалности

- добавяне на нов участник по отбор
- премахване на участник по име на отбор
- проверка дали даден отбор участва в лигата
- достъп до участник по име на отбор чрез `operator[]`

- добавяне на нов мач между два отбора
- добавяне на нов мач между два отбора с директно подаден резултат
- намиране на мач по идентификатор
- задаване на резултат за мач
- намиране на всички мачове на даден отбор
- намиране на броя изиграни мачове на даден отбор

- връщане на класирането(таблицата) на лигата
- намиране на лидера в лигата
- Класирането трябва да бъде сортирано по следните критерии в дадения ред:
  - точки
  - голова разлика
  - отбелязани голове
  - име на отбора

---

### Правила за точкуване
- победа — 3 точки
- равенство — 1 точка
- загуба — 0 точки

Головата разлика се изчислява като:

`отбелязани голове - допуснати голове`

---

#### Пример:

```cpp
int main() {
    League league("Premier League");

    auto liverpool = std::make_shared<Team>("Liverpool");
    auto arsenal = std::make_shared<Team>("Arsenal");
    auto chelsea = std::make_shared<Team>("Chelsea");
    auto united = std::make_shared<Team>("Manchester United");

    league.addParticipant(liverpool);
    league.addParticipant(arsenal);
    league.addParticipant(chelsea);
    league.addParticipant(united);

    league.addMatch("Liverpool", "Arsenal", 2, 1);
    league.addMatch("Chelsea", "Manchester United", 0, 2);
    league.addMatch("Liverpool", "Chelsea", 1, 1);
    league.addMatch("Arsenal", "Manchester United", 3, 2);

    if (const LeagueParticipant* participant = league["Manchester United"]) {
        std::cout << *participant << '\n';
    }

    std::cout << league << '\n';
}
